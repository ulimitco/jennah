# Jennah Frontend

Jennah Frontend is the main app and the user interface connected to Jennah API


## Installation

- Clone Git Source

```sh
git clone https://github.com/psalmeight/jn-frontend.git
cd jn-frontend
```

- Start via `npm`

```sh
npm start
```

- Start via `yarn`

```sh
yarn start
```
